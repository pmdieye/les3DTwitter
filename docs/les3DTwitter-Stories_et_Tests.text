Version 1

En tant qu'utilisateur je crée un compte

   - ouverture de la page d'accueil

   - click sur "s'inscrire"

   - formulaire inscription

   - saisie login, mot de passe puis validation

   - affichage page de bienvenue avec le nom de l’utilisateur, la date et l’heure

   - réception email de confirmation



En tant qu'utilisateur je me connecte:

    - ouverture de la page d'accueil

    - formulaire de connexion

    - saisie login, mot de passe puis validation

    - affichage page de bienvenue avec le nom de l’utilisateur, la date et l’heure


En tant qu'utilisateur je peux me deconnecter:
         - click sur bouton de déconnexion

   - affichage page d’accueil de l’application

Version 2

En tant qu'utilisateur  j'écris un message
    - click sur “écrire un message”
    - affichage d’une zone de texte pour écrire le message
    - sauvegarde automatique du message écrit en “état brouillon”

En tant qu'utilisateur j'envoie un message
    - sélectionner le message parmi les messages en état brouillon

-  click sur “envoyer message”
    - transmission du message
    - affichage d’une alerte “Message envoyé”  

En tant qu'utilisateur je reçois un message
    - affichage d’une arlette “Nouveau Message reçu”
    - affichage d’un bouton “lire message”
    - Click sur bouton “lire message”

- Ouverture du message

En tant qu'utilisateur  je supprime un message
    - affichage des messages
    - sélectionner le ou les messages à supprimer
    - click sur bouton “supprimer”
    - affichage alerte “message supprimé”

En tant qu'utilisateur je consulte l'historique des messages
    - click sur “message brouillon”
    - afficher une page avec la liste des messages brouillons

- click sur “message reçu”
    - afficher une page avec la liste des messages reçus

- click sur “message envoyé”
    - afficher une page avec la liste des messages envoyés

Entités

    - id
    - draft

- id

- received   
Message_sent
    - id
    - sent

Version 4

En tant qu’utilisateur je peux supprimer mon compte
        - click sur profil
        - ouverture page de profil
        - click bouton “suppression compte”
        - demande confirmation suppression
        - validation de la suppression du compte


En tant qu'utilisateur j'ajoute d'autres utilisateurs comme contact
    - click bouton “Ajouter un Contact”
    - affichage champ texte pour écrire le nom du contact à rechercher
    - click “rechercher”
    - affichage de la liste des personnes correspondantes au critère de recherche


En tant qu'utilisateur je consulte les messages de mes contacts


En tant qu'utilisateur je réponds aux messages de mes contacts
    - sélection du message auquel le user veut répondre
    - click sur “répondre”
    - affichage d’une zone de texte pour écrire le message
    - sauvegarde automatique du message écrit en “état brouillon”

-  click sur “envoyer message”
    - transmission du message
    - affichage d’une alerte “Message envoyé”
